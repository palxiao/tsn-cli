
## 个人node项目模板生成工具

### 安装
```
npm i -g tsn-cli
```
OR
```
yarn global add tsn-cli
```

### 快速初始化一个空项目，目前已有6套模板
```
tsn-cli init project-name
```

    ...

    cd project-name

    yarn

## Vue3 + vite

包含三个初始化模板，PC安装了ElementPlus，移动端安装了vant+css自适应

包含express环境，mysql连接，typescript开发体验及webpack打包
